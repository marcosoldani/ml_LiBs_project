# Platform-conditional pin: Apple Silicon Docker images SIGILL on
# cryptography >=42 because the wheel uses CPU instructions the Docker
# VM does not expose. On arm64 we therefore stay on 41.x and accept
# the residual CVEs (CVE-2023-50782 et al.) for local dev only — they
# are documented in CHANGELOG.md. On x86_64 (CI, Linux prod) we take
# the latest line, which closes every known advisory up to mid-2026.

cryptography>=46.0.7; platform_machine != "arm64" and platform_machine != "aarch64"
